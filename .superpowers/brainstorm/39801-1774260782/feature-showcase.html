Feature Showcase Style

We have 6 app screenshots (home, mood, gratitude, progress, activity, activities). How should the feature section present them?

A — Alternating left/right rows

Each feature gets its own row: screenshot on one side, headline + description on the other. Alternates direction. Great for storytelling a user journey.

B — Feature cards grid (3 per row)

Each feature is a white card with the screenshot thumbnail + label. 2 rows of 3. Quick, scannable, feels more like an app gallery.

C — Tabbed feature switcher

Pill tabs (Mood / Activities / Progress / Gratitude) that swap the screenshot + description shown. Interactive, compact, feels polished.
